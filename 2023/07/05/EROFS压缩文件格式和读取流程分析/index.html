EROFSеңЁеҝ«йҖҹиҝӯд»ЈдёӯеҮәзҺ°дәҶи®ёеӨҡж–°зү№жҖ§пјҢиҖҢж–ҮжЎЈе№¶дёҚдё°еҜҢгҖӮжң¬ж–ҮиҜ•еӣҫеҹәдәҺerofs-utilsзҡ„1.3зүҲжң¬ж‘ёжё…еҺӢзј©ж–Үд»¶еңЁEROFSдёӯзҡ„on-diskеӯҳеӮЁж јејҸе’Ңз”ЁжҲ·жҖҒж–Үд»¶иҜ»еҸ–зҡ„е…ій”®и·Ҝеҫ„пјҢе№¶ж №жҚ®commitдҝЎжҒҜзңӢжҮӮеҗҺз»ӯзүҲжң¬зҡ„жӣҙж–°гҖӮ

жң¬ж–Үдё»иҰҒе…іжіЁиҜ»еҸ–пјҲи§ЈеҺӢпјүзҡ„жөҒзЁӢпјҢеҹәжң¬дёҚж¶үеҸҠеҺӢзј©дҫ§зҡ„зү№жҖ§пјҲеҰӮdedupeпјүгҖӮ

еҺӢзј©ж–Үд»¶on-diskзҙўеј•

EROFSе°Ҷж–Үд»¶жңӘеҺӢзј©зҡ„еҺҹе§Ӣж•°жҚ®еҲ’еҲҶдёәиӢҘе№ІдёӘlcluster(Logical Cluster)пјҢеӣәе®ҡиҫ“еҮәеӨ§е°Ҹзҡ„еҺӢзј©ж•°жҚ®еҚ•е…ғеҲҷз§°дёәpcluster(Physical Cluster)гҖӮе…¶дёӯlclusterзҡ„еӨ§е°ҸжҳҜеӣәе®ҡзҡ„пјҢзӯүдәҺеқ—еӨ§е°Ҹ(4K)пјҢpclusterеҲҷеҸҜд»Ҙз”ұиӢҘе№ІдёӘиҝһз»ӯеқ—з»„жҲҗгҖӮж–Үд»¶зҙўеј•зҡ„д»»еҠЎе°ұжҳҜе°Ҷз»ҷе®ҡзҡ„йҖ»иҫ‘ең°еқҖпјҲеҢәй—ҙпјүзҝ»иҜ‘жҲҗзЈҒзӣҳдёҠзҡ„зү©зҗҶең°еқҖпјҲеҢәй—ҙпјүгҖӮеҜ№дәҺеҺӢзј©ж–Үд»¶пјҢEROFSж”ҜжҢҒLegacyе’ҢCompactedдёӨз§Қзҙўеј•ж јејҸпјҢж №жҚ®dinodeдёӯi_formatеӯ—ж®өеөҢе…Ҙзҡ„datalayoutеҠ д»ҘеҢәеҲҶгҖӮ

Legacy index

Legacyзҙўеј•зҡ„з»“жһ„еҰӮеӣҫжүҖзӨәгҖӮжҜҸдёӘinodeеңЁеҸҜйҖүзҡ„xattrеҢәеҹҹд№ӢеҗҺеӯҳеӮЁеҺӢзј©зҙўеј•пјҢе…¶дёӯй»„иүІйғЁеҲҶstruct z_erofs_map_headerжҳҜper-inodeзҡ„е…ғж•°жҚ®пјҢеҚ жҚ®8дёӘеӯ—иҠӮпјҢеҗҺйқўи·ҹзқҖ8еӯ—иҠӮзҡ„paddingгҖӮе®ғи®°еҪ•дәҶinodeеұӮйқўејҖеҗҜзҡ„еҺӢзј©зү№жҖ§(h_advice)пјҢдҪҝз”Ёзҡ„еҺӢзј©з®—жі•зұ»еһӢ(h_algorithmtype)е’Ңlclusterзҡ„еӨ§е°ҸпјҲh_clusterbitsпјҢзӣ®еүҚе®һзҺ°дёӯдёәеёёж•°пјүгҖӮ

зҙ§йҡҸе…¶еҗҺзҡ„жҳҜиӢҘе№ІдёӘ8еӯ—иҠӮз»“жһ„дҪ“struct z_erofs_vle_decompressed_indexпјҢе®ғжҳҜдёҖдёӘ4KB lclusterзҡ„зҙўеј•пјҢе®ҡд№үеҰӮдёӢпјҡ

е…¶дёӯпјҢdi_adviceи®°еҪ•дәҶlclusterзҡ„з§Қзұ»гҖӮPLAINзұ»еһӢзҡ„lclusterжңӘиў«еҺӢзј©пјҢд»ҘеҺҹе§Ӣж•°жҚ®зҡ„ж јејҸжҳ е°„еҲ°дёҖдёӘй•ҝеәҰдёә1дёӘblockзҡ„pclusterдёӯеҺ»пјӣHEADзұ»еһӢзҡ„lclusterжҳҜдёҖж®өextentзҡ„иө·е§ӢlclusterпјҢе®ғд»ЈиЎЁиҝҷдёӘextentдёӯзҡ„дёҖзі»еҲ—lclusterжҳ е°„еҲ°дәҶжҹҗдёӘpclusterпјҢе№¶еӯҳеӮЁдәҶpclusterзҡ„зү©зҗҶең°еқҖ(blk_addr)гҖӮиҜҘextentдёӯеү©дҪҷзҡ„lclusterеҲҷдёәNONHEAD lclusterпјҢе®ғи®°еҪ•дәҶдёҺеүҚдёҖдёӘHEAD lclusterе’ҢеҗҺдёҖдёӘHEAD lclusterзҡ„и·қзҰ»(delta[0]е’Ңdelta[1])гҖӮ

йңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢдёҖж®өextentе№¶дёҚдёҖе®ҡеҜ№йҪҗеҲ°lclusterзҡ„иҫ№з•ҢгҖӮиҝҷж„Ҹе‘ізқҖжҹҗдёӘlclusterзҡ„еүҚдёҖйғЁеҲҶе’ҢеҗҺдёҖйғЁеҲҶеҸҜиғҪд»ҺеұһдәҺдёҚеҗҢзҡ„extentгҖӮеңЁиҝҷз§Қжғ…еҶөдёӢпјҢиҝҷдёӘlclusterзҡ„зұ»еһӢжҳҜHEADпјҢе№¶дё”di_clusterofsжҳҜйқһ0еҖјпјҢи®°еҪ•дәҶextentиө·е§ӢдҪҚзҪ®еңЁиҜҘlclusterдёӯзҡ„еҒҸз§»йҮҸгҖӮ

дёҠеӣҫз»ҷеҮәдәҶдёҖдёӘдҫӢеӯҗгҖӮеӣҫдёӯlcluster0, lcluster1д»ҘеҸҠlcluster2зҡ„еүҚ1024еӯ—иҠӮз»„жҲҗдәҶз¬¬дёҖдёӘextentпјҢжҳ е°„еҲ°зү©зҗҶең°еқҖдёә0x233000пјӣlcluster2зҡ„еү©дҪҷйғЁеҲҶеұһдәҺз¬¬дәҢдёӘextentпјҢжҳ е°„еҲ°зү©зҗҶең°еқҖдёә0x234000зҡ„pclusterгҖӮдёӨдёӘpclusterзҡ„й•ҝеәҰеқҮдёә1дёӘеқ—еӨ§е°ҸгҖӮеӣ жӯӨlcluster0е’Ңlcluster2йғҪжҳҜHEAD lclusterпјҢдё”di_clusterofsеҲҶеҲ«дёә0е’Ң1024гҖӮlcluster1жҳҜNONHEAD lclusterпјҢе№¶дё”дёҺеүҚеҗҺHEAD lclusterзҡ„и·қзҰ»йғҪжҳҜ1гҖӮ

Big pcluster

еңЁmap_blockзҡ„ж—¶еҖҷпјҢдҪңдёәеҺӢзј©ж–Үд»¶зі»з»ҹпјҢEROFSиҰҒеҗҢж—¶з»ҷеҮәextentзҡ„llenе’ҢplenгҖӮж—©жңҹзүҲжң¬зҡ„EROFSиҰҒжұӮpclusterзҡ„еӨ§е°ҸзӯүдәҺеқ—еӨ§е°Ҹ(4KB)пјҢеӣ жӯӨеҸҜд»ҘеҒҮи®ҫplenзҡ„й•ҝеәҰе°ұжҳҜ4KBгҖӮLinux v5.13еј•е…ҘдәҶbig pclusterзү№жҖ§пјҢе®ғе…Ғи®ёpclusterзҡ„еӨ§е°Ҹдёәеқ—еӨ§е°Ҹзҡ„еҖҚж•°гҖӮдәҺжҳҜпјҢжҲ‘д»¬йңҖиҰҒеңЁж–Үд»¶зҙўеј•дёӯи®°еҪ•иҝҷдёӘеӨ§е°ҸгҖӮ

EROFSйҖүжӢ©е°ҶpclusterеӨ§е°Ҹи®°еҪ•еңЁз»„жҲҗextentзҡ„з¬¬дёҖдёӘNONHEAD lclusterдёӯгҖӮиҝҷдёӘlclusterиў«з§°дёәCBLKCNT lclusterпјҢе®ғзҡ„delta[0]дёҖе®ҡжҳҜ1пјҢеӣ жӯӨеҸҜд»ҘеӨҚз”Ёdelta[0]зҡ„з©әй—ҙгҖӮиҝҷз§Қжғ…еҶөдёӢпјҢdelta[0]дёӯзҡ„Z_EROFS_VLE_DI_D0_CBLKCNTиў«и®ҫдёә1гҖӮеҰӮжһңжҹҗдёӘextentжІЎжңүNONHEAD lclusterпјҢиҜҙжҳҺе®ғеҜ№еә”зҡ„pclusterй•ҝеәҰдёҖе®ҡеҸӘиғҪжҳҜ1дёӘеқ—еӨ§е°ҸгҖӮ

еҰӮеӣҫжүҖзӨәпјҢз”ұдәҺз¬¬дёҖдёӘpclusterзҡ„еӨ§е°ҸжҳҜ8192пјҲ2дёӘеқ—еӨ§е°ҸпјүпјҢеӣ жӯӨеҜ№еә”extentдёӯзҡ„з¬¬дёҖдёӘNONHEAD lclusterзҡ„delta[0]и®°еҪ•дәҶ0x2 | 1 << (Z_EROFS_VLE_DI_D0_CBLKCNT)гҖӮ

Compacted Index

Legacy Indexзҡ„дёҖеӨ§зјәзӮ№жҳҜеҜ№дәҺжҜҸдёӘ4KBеӨ§е°Ҹзҡ„lclusterпјҢйғҪиҰҒи®°еҪ•дёҖдёӘ8еӯ—иҠӮзҡ„зҙўеј•пјҢеҜјиҮҙдәҶдёҚеҝ…иҰҒзҡ„з©әй—ҙеҚ з”ЁгҖӮCompacted Indexзҡ„ж ёеҝғжҖқжғіе°ұжҳҜеҺӢзј©ж–Үд»¶зҙўеј•жүҖеҚ з”Ёзҡ„з©әй—ҙгҖӮEROFSжҠҠзӣёйӮ»иӢҘе№ІдёӘlclusterзҡ„зҙўеј•зј–з ҒеңЁдёҖиө·пјҢз§°дёәpackгҖӮжҜҸдёӘpackйңҖиҰҒйў„з•ҷ4еӯ—иҠӮжқҘеӯҳж”ҫжҹҗдёҖдёӘextentеҜ№еә”зҡ„иө·е§Ӣpclusterең°еқҖпјҢз”ұдәҺеҚ•дёӘж–Үд»¶зҡ„pclusterжҳҜиҝһз»ӯзҡ„пјҢе…¶д»–extentзҡ„зү©зҗҶең°еқҖеҸҜд»Ҙж №жҚ®иҜҘең°еқҖжҺЁз®—еҮәжқҘгҖӮ

е…·дҪ“жқҘиҜҙпјҢEROFSж”ҜжҢҒcompacted_2Bе’Ңcompacted_4BдёӨз§Қзҙўеј•ж јејҸпјҢеҪ’зәҰеҗҺжҜҸдёӘlclusterзҙўеј•еҲҶеҲ«йңҖиҰҒеҚ з”Ё2еӯ—иҠӮе’Ң4еӯ—иҠӮзҡ„еӯҳеӮЁз©әй—ҙпјҡ

  • 2B: жҜҸдёӘlclusterзҙўеј•еҢ…еҗ«14дёӘдҪҚпјҢжҜҸ16дёӘзҙўеј•дёәдёҖдёӘpackпјҢpackеӨ§е°Ҹдёә\((14 * 16 + 32) / 8 = 32\)еӯ—иҠӮгҖӮ
  • 4B: жҜҸдёӘlclusterзҙўеј•еҢ…еҗ«16дёӘдҪҚпјҢжҜҸ2дёӘзҙўеј•дёәдёҖдёӘpackпјҢpackеӨ§е°Ҹдёә\((16 * 2 + 32) / 8 = 8\)еӯ—иҠӮгҖӮ

еҸҜи§ҒпјҢ2Bе’Ң4Bзҡ„зҙўеј•ж јејҸеҲҶеҲ«йңҖиҰҒ32еӯ—иҠӮе’Ң8еӯ—иҠӮеҜ№йҪҗпјҢд»ҘдҝқиҜҒе®ғд»¬дёҚдјҡи·Ёи¶Ҡеқ—иҫ№з•ҢгҖӮеӣ жӯӨEROFSдҪҝз”Ё2Bзҙўеј•дҪңдёәcompacted indexзҡ„дё»дҪ“йғЁеҲҶпјҢдҪҝз”Ё4Bзҙўеј•дҪңдёәеүҚеҗҺеЎ«е……жқҘж»Ўи¶і32еӯ—иҠӮзҡ„еҜ№йҪҗиҰҒжұӮгҖӮ

14дҪҚlclusterзҙўеј•зҡ„дҪҝз”ЁпјҡдёҖе…ұжңү3з§ҚдёҚеҗҢзұ»еһӢзҡ„lclusterпјҢеӣ жӯӨйңҖиҰҒ2дёӘдҪҚжқҘеӯҳеӮЁе…¶зұ»еһӢгҖӮеҜ№дәҺHEADе’ҢPLAIN lclusterпјҢеү©дҪҷ12дёӘдҪҚз”ЁжқҘеӯҳеӮЁclusterofsпјӣеҜ№дәҺNONHEAD lclusterпјҢеӯҳеңЁ3з§Қжғ…еҶөпјҡ

  1. CBLKCNT lclusterпјҡеӯҳеӮЁpclusterеӨ§е°ҸгҖӮ
  2. жҒ°еҘҪдҪҚдәҺдёҖдёӘpackзҡ„жңҖеҗҺпјҡеӯҳеӮЁdelta[1]гҖӮ
  3. е…¶д»–жғ…еҶөпјҡеӯҳеӮЁdelta[0]гҖӮ

жҜҸдёӘpackдёӯеҸӘдјҡеӯҳдёҖдёӘpclusterзҡ„ең°еқҖпјҡ

  • еҪ“дёҚејҖеҗҜbig pclusterзү№жҖ§ж—¶пјҢиҜҘең°еқҖдёәpackдёӯз¬¬дёҖдёӘHEAD/PLAIN lclusterжүҖеҜ№еә”зҡ„pclusterең°еқҖеҮҸеҺ»дёҖдёӘеқ—еӨ§е°ҸгҖӮ
  • еҪ“ејҖеҗҜbig pclusterзү№жҖ§ж—¶пјҢиҜҘең°еқҖдёәpackдёӯз¬¬дёҖдёӘж–°ејҖеҗҜзҡ„extentжүҖеҜ№еә”зҡ„pclusterең°еқҖгҖӮеҜ№дәҺдёҖиҲ¬extentпјҢж–°ејҖеҗҜеҚіHEAD/PLAIN lclusterиҗҪеңЁиҜҘpackдёӯпјӣеҜ№дәҺжҳ е°„еҲ°big pclusterзҡ„extentпјҢж–°ејҖеҗҜеҚіCBLKCNT lclusterпјҲиҖҢдёҚжҳҜHEAD lclusterпјүиҗҪеңЁиҜҘpackдёӯгҖӮ

v1.4

fiemap

еңЁдёҖиҲ¬зҡ„и§ЈеҺӢжөҒзЁӢдёӯпјҢеҰӮжһңread()иҜ»еҸ–зҡ„жңҖеҗҺдёҖдёӘextentжҳҜдёҚе®Ңж•ҙзҡ„пјҢEROFSеҸҜд»ҘйғЁеҲҶи§ЈеҺӢpclusterжқҘж»Ўи¶іиҜ»еҸ–иҜ·жұӮпјҢиҖҢдёҚйңҖиҰҒи®Ўз®—иҜҘextentзҡ„е®Ңж•ҙй•ҝеәҰгҖӮеҰӮеӣҫжүҖзӨәпјҢй»„иүІйғЁеҲҶж ҮиҜҶдәҶз¬¬дёҖж¬Ўиҝ”еӣһзҡ„map->m_laе’Ңmap->llenпјҢе®ғе№¶дёҚжҳҜдёҖдёӘе®Ңж•ҙзҡ„extentгҖӮ

з„¶иҖҢпјҢеӯҳеңЁдёҖдёӘioctlйҖүйЎ№fiemapпјҢе®ғд»Һз”ЁжҲ·жҖҒиҺ·еҸ–ж–Үд»¶зҡ„extentгҖӮдәҺжҳҜEROFSеј•е…ҘдәҶz_erofs_get_extent_decompressedlen()еҮҪж•°пјҢз”ЁдәҺеңЁиҝҷз§Қжғ…еҶөдёӢи®Ўз®—е®Ңж•ҙзҡ„extentй•ҝеәҰгҖӮе®ғеңЁеёҰжңүEROFS_GET_BLOCKS_FIEMAP flagзҡ„z_erofs_do_map_blocks()дёӯиў«и°ғз”ЁгҖӮ

еҜ№дәҺlegacyзҙўеј•пјҢextentй•ҝеәҰеҸҜд»Ҙз®ҖеҚ•д»ҺеӯҳеӮЁзҡ„delta[1]дёӯиҺ·еҸ–пјҢдёҚйңҖеј•е…Ҙд»»дҪ•йўқеӨ–йҖ»иҫ‘гҖӮз„¶иҖҢеҜ№дәҺcompactedзҙўеј•пјҢеҸӘжңүжҜҸдёӘpackзҡ„жңҖеҗҺдёҖдёӘlclusterжүҚдјҡеӯҳеӮЁdelta[1]пјҢеӣ жӯӨиҝҷдёӘиҝҮзЁӢйңҖиҰҒеҗ‘еҗҺйҒҚеҺҶжҜҸдёӘlclusterзҡ„зҙўеј•пјҢзӣҙеҲ°жүҫеҲ°дёҖдёӘHEAD/PLAIN lclusterдёәжӯўгҖӮйҒҮеҲ°жңүж•Ҳзҡ„delta[1]еҸҜд»ҘеҠ йҖҹйҒҚеҺҶиҝҮзЁӢгҖӮ

v1.5

е®һзҺ°дёҠпјҢper inodeзҡ„struct z_erofs_map_headerзҡ„h_adviceдјҡи®°еҪ•ztalipackingжҳҜеҗҰејҖеҗҜгҖӮжӯӨеӨ–пјҢиҜҘз»“жһ„дҪ“иҝҳз»ҙжҠӨдәҶдёҖдёӘh_idata_sizeеҸҳйҮҸпјҢеӯҳеӮЁеҶ…иҒ”йғЁеҲҶж•°жҚ®зҡ„еҺӢзј©еҗҺеӨ§е°ҸгҖӮ

EROFSе…Ғи®ёе°ҶиӢҘе№ІдёӘж–Үд»¶зҡ„дёҖйғЁеҲҶжү“еҢ…еңЁдёҖдёӘpacked_fileдёӯдёҖиө·еҺӢзј©пјҢд»ҘжӯӨжҸҗй«ҳеҺӢзј©зҺҮгҖӮиҝҷдәӣйғЁеҲҶз§°дёәfragmentгҖӮдёҖиҲ¬жқҘиҜҙпјҢж–Үд»¶зҡ„tailйғЁеҲҶдјҡдҪңдёәfragmentпјҢд№ҹеҸҜд»Ҙе°ҶдёҖдёӘе®Ңж•ҙзҡ„ж–Үд»¶дҪңдёәfragmentгҖӮ

е®һзҺ°дёҠпјҢper inodeзҡ„struct z_erofs_map_headerзҡ„h_adviceдјҡи®°еҪ•иҜҘinodeжҳҜеҗҰеҢ…еҗ«fragmentгҖӮеҰӮжһңжңүзҡ„иҜқпјҢfragmentзҡ„ж–Үд»¶еҒҸз§»йҮҸиў«и®°еҪ•еңЁh_fragmentoffдёӯгҖӮдёҖдёӘж–Үд»¶дёҚеҸҜиғҪеҗҢж—¶еҢ…еҗ«fragmentе’ҢејҖеҗҜztailpackingзү№жҖ§пјҢеӣ жӯӨh_fragmentoffе’Ңh_idata_sizeдҪҚдәҺеҗҢдёҖдёӘunionдёӯгҖӮ

z_erofs_do_map_blocks()дёӯпјҢеҰӮжһңеҸ‘зҺ°еҪ“еүҚиҜ»еҸ–зҡ„extentжҳҜдёҖдёӘfragmentпјҢдјҡи®ҫзҪ®EROFS_MAP_FRAGMENTж Үи®°гҖӮz_erofs_read_one_data()зңӢеҲ°иҜҘж Үи®°еҗҺпјҢдјҡеҺ»packed_inodeдёӯж №жҚ®и®°еҪ•зҡ„fragmentoffиҜ»еҸ–ж•°жҚ®гҖӮ
